# Contract IQ — Project Roadmap

## Vision
- Deliver a contract intelligence tool that extracts key terms, flags risks, and summarizes obligations.

## Milestones (Q4 2025–Q2 2026)
- M1: Define MVP scope and data model
- M2: Build ingestion and parsing pipeline
- M3: Key term extraction (NLP baseline)
- M4: Risk flags and obligations summary
- M5: UI prototype and user testing
- M6: Performance tuning and deployment plan
- ## Dependencies
- spaCy or transformers for NLP
- Postgres for storage
- Streamlit/Flask for UI

## Risks
- Ambiguity in contracts
- Data privacy/security
- Model generalization across industries
